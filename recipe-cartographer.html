how to generate the file
Run in your bitbake build dir (where you'd normally build):
bitbake -g <image-or-recipe> That writes task-depends.dot, pn-buildlist, and recipe-depends.dot to the current dir. Drop the task-depends.dot here.

Examples:
bitbake -g <your-image> — whole image graph
bitbake -g <recipe> — one recipe's build closure

Notes: this view shows build-time (DEPENDS) edges, the porting-relevant graph, not runtime RDEPENDS. Set the root recipe on the left to whatever recipe you want at the centre of the graph; it re-roots the same file without re-dropping. Layer/team colours are read from each recipe's .bb path in the labels.
